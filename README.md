# ContaBlog

**A comprehensive accounting and finance blog featuring technical articles, tax regulations (NIF/NIIF), and interactive financial calculators for students and professionals.**

## 🚀 Overview
ContaBlog is a modern web platform designed to provide excellence in accounting education. Developed by students at UJAT, it offers expert insights into financial regulations and practical tools for everyday accounting tasks.

## ✨ Features
- **Technical Articles**: In-depth analysis of NIF, NIIF, and inflation accounting.
- **Interactive Tools**: Automated calculators for ISR and provisions.
- **Modern Design**: Built with a focus on UI/UX using Vanilla CSS and clean JavaScript.
- **SEO Optimized**: Following best practices for academic visibility.

## 🛠️ Tech Stack
- HTML5
- Vanilla CSS
- JavaScript
- Vite (Build Tool)
